# Small Tools

A collection of utility scripts for daily development tasks, including API interaction, data processing, and automation workflows.

## Included Tools

| Script | Description |
|--------|-------------|
| `build_dify_ready_payload.py` | Build and prepare Dify API payloads |
| `create_ragflow_agent.py` | Create and configure RAGFlow agents |
| `ragflow_update_fact_cards.py` | Update fact cards in RAGFlow knowledge base |
| `optimize_dify_ready_payload_v2.py` | Optimize Dify payload configurations |

## Tech Stack
- **Language:** Python
- **APIs:** RAGFlow, Dify

## Usage
Each script can be run independently. Check the source for required environment variables and parameters.

source_files=47
output_dir=C:\Users\wangl\School\dify_ready_payload
